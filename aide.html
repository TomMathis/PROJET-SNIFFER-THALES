Comment fonctionne le décodeur de trame ?

Objectifs

L’objectif du projet est de récupérer et analyser des trames Ethernet d'un satellite afin d'en tirer les informations voulues pour ensuite les publier sur un site web avec lequel on pourra voir les données du test souhaité. On doit pouvoir modifier les valeurs et si on veut, pouvoir les extraire dans un fichier Excel, qui contiendra la base de données des trames.

Le but est de parvenir à décoder un fichier contenant des trames en binaire extraites d’un satellite grâce à un programme python qui prendra en paramètre le fichier contenant les trames binaires. Ensuite, le programme devra extraire les informations voulues et les stocker sur la base de données. L’utilisateur doit pouvoir accéder aux trames par rapport à la date d’exécution, modifier les noms des champs, créer un fichier csv ou Excel regroupant la base de données et aussi avoir la possibilité de supprimer les trames en fonction de la date d’exécution.


Fonctionnement du site Web

ETAPE 1 : Exécuter le programme python "main.py" ce qui permettra d'insérer les trames du test dans la base de données (phpMyAdmin)
ETAPE 2 : Rafraichir le site Web et sélectionner le test
ETAPE 3 : Choisir la bonne date d'exécution du test ce qui permettra de filtrer et d'afficher moins de ligne
ETAPE 4 : Cliquer sur le bouton "Voir les trames"

Vous pouvez également supprimer le test correspondant à la date d'exécution sélectionné, renommer les champs de la base de données et même exporter en format Excel le tableau affiché.
